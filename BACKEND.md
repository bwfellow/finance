# Backend Structure (`convex/`)

The backend uses [Convex](https://convex.dev) and follows a layered architecture:

- **`functions.ts`** - Public API (queries/mutations) with auth checks
- **`model.ts`** - Data access and business logic
- **`shared/`** - Reusable validators

Each domain module lives in its own folder with those two files.

---

## Core Files

| File | Purpose |
|---|---|
| `schema.ts` | Database schema. Defines 4 tables (`categories`, `categoryBudgets`, `transactions`, `incomeEntries`) plus auth tables. Each table is scoped by `userId` with compound indexes on user/month. |
| `auth.config.ts` | Auth provider config. Reads domain from `CONVEX_SITE_URL` env var. |
| `auth.ts` | Sets up Password and Anonymous auth providers. Exports `loggedInUser` query to fetch the current user. |
| `http.ts` | Attaches auth HTTP routes to the router. Default export used by Convex. |
| `router.ts` | Initializes the `httpRouter` instance. |

---

## Shared

| File | Purpose |
|---|---|
| `shared/validators.ts` | Reusable Convex validators: `monthValidator` (`"YYYY-MM"`), `dateValidator` (`"YYYY-MM-DD"`), `amountValidator`, `categoryNameValidator`, `descriptionValidator`, `targetTypeValidator`, and composite field sets (`targetFields`, `transactionFields`, `incomeFields`). |

## Domain Modules

### `budget/`

The read-only aggregation layer. No mutations — it pulls data from all other modules.

| File | Exports | Details |
|---|---|---|
| `functions.ts` | `getBudgetData` (query) | Takes a `month` param, returns full budget view for the logged-in user. |
| `model.ts` | `getBudgetData` | Aggregates categories, budgets, transactions, and income. Calculates per-category assigned/activity/available, target progress, rollover balances, and "ready to assign" (total income - total assigned). Handles 3 target types: `monthly_spending`, `monthly_savings`, `balance_by_date`. |

### `categories/`

Budget category management (e.g. "Groceries", "Rent").

| File | Exports | Details |
|---|---|---|
| `functions.ts` | `createCategory`, `deleteCategory`, `setTarget`, `removeTarget` (mutations) | Create auto-assigns sort order. Delete is a soft-delete (hides the category). |
| `model.ts` | `getCategoriesForUser`, `getVisibleCategoriesForUser`, `insertCategory`, `getCategory`, `hideCategory`, `setTarget`, `removeTarget` | Visible categories sorted by `sortOrder`. Targets have optional `targetType`, `targetAmount`, and `targetDate`. |

### `categoryBudgets/`

Monthly budget allocations per category (how much you assign to "Groceries" in January, etc).

| File | Exports | Details |
|---|---|---|
| `functions.ts` | `assignBudget` (mutation) | Takes `categoryId`, `month`, `amount`. Upserts the record. |
| `model.ts` | `getBudgetsForUserAndMonth`, `getAllBudgetsForUser`, `getBudgetForCategoryAndMonth`, `upsertBudget` | Tracks cumulative assignments per category per month. |

### `income/`

Income entries (paychecks, freelance payments, etc).

| File | Exports | Details |
|---|---|---|
| `functions.ts` | `addIncome`, `deleteIncome` (mutations) | Auto-extracts month from date. |
| `model.ts` | `getIncomeForUserAndMonth`, `getAllIncomeForUser`, `insertIncome`, `getIncome`, `deleteIncome` | Each entry stores `userId`, `month`, `description`, `amount`, `date`. |

### `transactions/`

Spending transactions tied to a category.

| File | Exports | Details |
|---|---|---|
| `functions.ts` | `addTransaction`, `deleteTransaction` (mutations) | Auto-extracts month from date. Links to a category. |
| `model.ts` | `getTransactionsForUserAndMonth`, `getAllTransactionsForUser`, `insertTransaction`, `getTransaction`, `deleteTransaction` | Each entry stores `userId`, `categoryId`, `month`, `description`, `amount`, `date`. |

---

## `_generated/`

Auto-generated by Convex. Do not edit. Contains typed API bindings (`api.d.ts`, `api.js`), data model types (`dataModel.d.ts`), and server helpers (`server.d.ts`, `server.js`).

## Folder Tree

```
convex/
├── _generated/          # Auto-generated Convex types & bindings
├── budget/
│   ├── functions.ts     # getBudgetData query
│   └── model.ts         # Budget aggregation & calculations
├── categories/
│   ├── functions.ts     # CRUD mutations for categories
│   └── model.ts         # Category data access
├── categoryBudgets/
│   ├── functions.ts     # assignBudget mutation
│   └── model.ts         # Budget allocation data access
├── income/
│   ├── functions.ts     # addIncome, deleteIncome mutations
│   └── model.ts         # Income data access
├── shared/
│   └── validators.ts    # Reusable field validators
├── transactions/
│   ├── functions.ts     # addTransaction, deleteTransaction mutations
│   └── model.ts         # Transaction data access
├── auth.config.ts       # Auth provider configuration
├── auth.ts              # Auth setup (Password + Anonymous)
├── http.ts              # HTTP router with auth routes
├── router.ts            # httpRouter instance
├── schema.ts            # Database schema & indexes
└── tsconfig.json        # TypeScript config for Convex
```
